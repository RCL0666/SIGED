# SIGED

Pagina web estatica para gestion de alumnos con Bootstrap.

## Como abrir

Abre `index.html` directamente en el navegador.

Tambien puedes levantar un servidor local:

```bash
node server.js
```

Luego abre `http://127.0.0.2:5500/index.html`.

Nota: en este equipo `127.0.0.1:5500` esta ocupado por otro servicio, por eso Live Server queda configurado para usar `127.0.0.2:5500`.

## Funciones

- Vista de profesor y vista de alumno.
- Asignacion y eliminacion de notas.
- Consulta de notas por alumno.
- Registro diario de asistencia.
- Eventos para un alumno o para todos.
- Datos demo guardados en `localStorage`.

Usa el boton **Reiniciar demo** para volver a los datos iniciales.
